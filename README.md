## INSTALLATION

This is documented as a clear guide of how to install and configure **Parrot OS** in the **Oracle VM virtualbox**. Parrot OS is a Linux based distribution aimed at **security, privacy, and development** and packed with a large number of pre-installed penetration testing, reverse engineering, and digital forensics tools.
The installation guide includes:
- System requirements
- Virtual machine configuration
- Step-by-step installation procedure
- Common issues encountered during installation and their solutions
- Network and display configuration tips

This section demonstrates practical understanding of operating system installation in a virtualized environment.
